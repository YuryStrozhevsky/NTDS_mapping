# Basic Information
The project is about making a full list of mapping between _lDAPDisplayName_ and names of columns inside _datatable_ in **NTDS.DIT** file. In order to make the list just put **NTDS.DIT** file in current folder.

All the project written using pure C++ and ESE (Extended Storage Engine) API. Example file with such mapping for Windows Server 2019 you can find in [**this file**](https://github.com/YuryStrozhevsky/NTDS_mapping/blob/main/ATTR_MAPPING.txt).
